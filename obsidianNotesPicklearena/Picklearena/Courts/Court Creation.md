To create a Group the user need to enter the following info:

- Upload a banner
- Court Name
- Location
- Court Description

![[photo_2026-01-16_12-33-00.jpg]]

After clicking the "Continue" button the user is redirected to the second step of Court Creation process where the user need to clarify the following info:

- Working days of court
- Working hours of court

![[photo_2026-01-16_12-33-01.jpg]]

After clicking the "Continue" button the user is redirected to the third step of Court Creation process where the user need to clarify the following info:

- Player limit
- Court privacy
