Once user signed up they see banner in their profile with request to fill profile info and also percentage of filled profile near the avatar 

There are 5 steps to fill profile: 
1. Upload avatar and cover 
2. Fill in player preferences 
3. Fill in time preferences (days and hours) 
4. Fill in gear 
5. Fill in interests 

First Step: User is able to upload avatar and cover for te profile 


Second Step: User can set preferences, such as: 
Best Hand (Right, Left, Both) 
Match Type (Competitive, Casual, Both) 
Court position (Left, Righ, No preferences) 
Playstyle (Net Player, Baseline Player, All Rounder) 
How ofter do you play (1-2/Month, 1-2/Week, 3-5/Week, Daily)
