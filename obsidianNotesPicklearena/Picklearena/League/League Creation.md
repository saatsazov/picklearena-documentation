To create a League the user need to enter the following info:

- Upload a banner
- League Name
- Location
- Description (User can pick one of pregenerated League description)

![[photo_2026-01-13_14-32-37.jpg]]

After clicking the "Continue" button the user is redirected to the second step of League Creation process where the user need to clarify the following info:

- [[Skill Level]]
- [[Age of opponent]]
- Number of Teams
- Visibility (Others can find and join, only invited people can join)
- Match Format (Single, Best of 3)
- Type of league (Free or Money)
- Make it private (Yes, No)

![[photo_2026-01-13_14-32-39 1.jpg]]

If league was set to Money the user needs to enter additional info:

- Price of admission
- Winning share
