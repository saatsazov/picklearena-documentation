Teams can join or create a league. [[League Creation]]
League can be searched by using search input field.

Leagues can be filtered by the following criterias:
- Location range
- [[Skill Level]]
- [[Age of opponent]]

By clicking on the league the user is redirected to the [[League's page]]
Also leagues can be checked on the map.
