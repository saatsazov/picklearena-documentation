Each user can check the league feed if the league is public.

Once user navigated to the league's page new 4 tabs appears:

- League (Active Tab)
- Matches (Match history)
- Teams (Teams that are in league)
- Scores (Leader board of a league)

![[photo_2026-02-09_09-03-12.jpg]]

Each user can Join or Report a league. To join league the user needs to select a team that will join the league.

If the user is admin of a league they can perform the following actions after clicking the Manage button:

- Finish league
- Share league
- Edit league photo
- Change league privacy
- Change league visibility
- Edit general league information added during the [[League Creation]]
- Check administrators
- Check teams
- Delete league

![[photo_2026-02-09_09-03-13.jpg]]

Teams canleave the league by clicking on the "Joined as 'team name'" button.
Every user by navigating to the league page and clicking the "..." button can report a league. 
Also when the user clicks on the "..." button near Join Button they can Invite others to league or share a league.
