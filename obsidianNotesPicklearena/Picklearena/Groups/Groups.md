Users can join or create a group. [[Group Creation]]
Group can be searched by using search input field.

Group can be filtered by the following criterias:
- Location range
- [[Skill Level]]
- [[Age of opponent]]

The user have possibility to navigate to the following tabs:

- My groups
- All groups

By clicking on the group the user is redirected to the [[Group's page]]
Also groups can be checked on the map using the "Show on Map" button.
