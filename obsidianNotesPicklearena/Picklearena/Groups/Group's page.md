Each user can check the group feed if the group is public.

Once user navigated to the group's page new 3 tabs appears:

- Group (Active Tab)
On the group tab the user can check the group's feed with [[Posts]]
- Events ([[Event]] list)
By clicking on the "Create new event" user is redirected to the [[Event Creation]] page
- Members (Members list)

![[photo_2026-01-14_12-00-15.jpg]]

Each user can Join or Report a Group. To join group the user needs to click the "Join Group" button

If the user is admin of a group they can perform the following actions after clicking the Manage button:

- Invite users to group
- Share group
- Edit group photo
- Change group privacy
- Change group visibility
- Edit general group information added during the [[Group Creation]]
- Check administrators
- Check members
- Check notifications
- Leave group
- Delete group

![[photo_2026-01-14_12-00-17.jpg]]

Every user by navigating to the group page and clicking the "..." button can report a group.

When the admin navigates to the Group after it's creation the banner is shown that redirects to the [[Event Creation]] page.
