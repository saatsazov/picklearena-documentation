To create a Group the user need to enter the following info:

- Upload a Profile Picture
- Upload a banner
- Group Name
- Location
- Description (User can pick one of pregenerated League description)

![[photo_2026-01-14_11-40-59.jpg]]

After clicking the "Continue" button the user is redirected to the second step of Group Creation process where the user need to clarify the following info:

- [[Skill Level]]
- [[Age of opponent]]
- Visibility (Others can find and join, only invited people can join)
- Make it private (Yes, No)
