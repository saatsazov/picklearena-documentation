
Once user navigated to the events's page new 2 tabs appears:

- Members (Members list)
- Games (Games list of the event)

![[photo_2026-01-14_12-32-16.jpg]]

Each user can Join an Event if it's Public. To join the event the user needs to click the "Join Event" button

If the user is admin of an event they can perform the following actions after clicking the Manage button:

- Invite users to event
- Share event
- Change event privacy
- Edit general event information that was added during the [[Event Creation]]
- Check administrators
- Check participants
- Leave event
- Delete event
