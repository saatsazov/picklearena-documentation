To create an Event the user need to enter the following info:

- Upload a banner
- Event Name
- Location
- Description

![[photo_2026-01-14_12-22-38.jpg]]

After clicking the "Continue" button the user is redirected to the second step of Event Creation process where the user need to clarify the following info:

- Start Date & time
- [[Repeat Event]]
- Type of Event (Free or Money)
- If Money type was selected user need also add the price of entering the event
- [[Player Limit]]
- Make it private (Yes, No)
