By clicking the "Recent Activity" input field button the user is redirected to the [[Post Creation]] page.

The user can perform the following actions on the posts:

- Like a post
- Unlike a post
- Comment a post
- Like a comment
- Report a comment
- Delete a comment (if the comment was left by the user)
- Block User
- Report Post
- Delete post (if post was created by the user)
