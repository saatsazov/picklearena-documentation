When the user navigates to the Match Details page they can check the following info:
- Score of the match
- Teams that played a match
- Status of a match
- Team members for each team
- Report a match
- For own match the user can edit a score

By clicking on the team member the user is being redirected to the [[Profile of another user]]
By clicking on the team the user is being redirected to the [[Team's page]]
