User can create a chat with another users. Chat can be created via the [[Profile of another user]] by clicking on the 'Message' button. Or by clicking on the '+' icon on the Chats page.

User can search for a chat by users.

Also, user can create a group by clicking on the '+' icon and choose the corresponding option ('Create group chat')

![[photo_2026-02-17_15-57-42.jpg]]

In chat user can send text message and add the following attachments:
- Files
- Camera
- Photos
- Poll
