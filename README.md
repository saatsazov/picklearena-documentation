# Picklearena Documentation

This repository contains comprehensive documentation for the Picklearena mobile application - a social platform for pickleball players to find courts, organize events, join leagues, and connect with other players.

## About Picklearena

Picklearena is a mobile app designed to help pickleball enthusiasts:
- Find and book courts
- Create and join events
- Form teams and join leagues  
- Connect with other players through groups and posts
- Manage profiles and skill levels
- Chat and coordinate matches

## Documentation Structure

The documentation is organized using Obsidian vault format with interconnected markdown files and screenshots. All documentation files are located in the `obsidianNotesPicklearena/Picklearena/` directory.
